* Author: [Stefan Touhami](https://touhami.pl)

## Requirements
* [Git](https://git-scm.com/downloads)
* [Docker](https://docs.docker.com/install) and [Docker Compose](https://docs.docker.com/compose/install)
* WeatherApi Key
